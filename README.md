# Лекало ткани для SketchUp

Расширение SketchUp для подготовки мебельных лекал и карт раскроя ткани.

Автор: Малкаров Сослан

## Возможности

- создание деталей из выбранных граней, групп и компонентов;
- развертка связной лицевой поверхности в одну деталь;
- припуски, линии реза, насечки и направление ворса;
- ручная и автоматическая раскладка без наложения деталей;
- свободная навигация по карте раскроя колесом и средней кнопкой мыши;
- экспорт SVG и XML;
- обновление из локального RBZ/RB или последнего GitHub Release.

## Установка

### RBZ

1. Откройте в SketchUp `Окно -> Диспетчер расширений`.
2. Нажмите `Установить расширение`.
3. Выберите `lekalo_cutting.rbz`.

### Windows

Запустите `LekaloCuttingSetup_<версия>.exe` и выберите установленные версии SketchUp.

## Сборка

```powershell
powershell -ExecutionPolicy Bypass -File scripts/build.ps1
```

Готовые файлы создаются в папке `dist`.

## Обновления

Кнопка обновления на панели инструментов предлагает два режима:

- `С сервера` загружает последний RBZ из GitHub Releases;
- `Из файла` устанавливает выбранный RBZ или заменяет отдельный RB-файл.

После обновления необходимо перезапустить SketchUp.
